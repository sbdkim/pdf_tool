# PDF Foundry

PDF Foundry is a browser-only PDF toolkit focused on reliable local workflows instead of broad but shallow feature sprawl.

## Launch promise
- Fast local PDF tools with no uploads
- Professional workspace for merge, split, rotate, reorder, and delete
- Public docs, roadmap, FAQ, privacy copy, and support hooks included

## Run locally
```bash
npm install
npm run dev
```

## Quality checks
```bash
npm run lint
npm run typecheck
npm run test
npm run build
```

## GitHub Pages build
```bash
PAGES_BASE=/pdf_tool/ npm run build
```

## Public pages
- `/` landing page
- `/app.html` workspace
- `/faq.html` FAQ
- `/privacy.html` privacy
- `/roadmap.html` roadmap
- `/terms.html` terms
